# getExe-replacer

Заменяет способ получения пути к пакету с прямого обращения через pkgs.<package>/bin/<command> на получение пути через функцию lib.getExe

Для получения пути к другому исполняемому файлу в пакете используется lib.getExe'

Таким образом производит преобразование:

```nix
${pkgs.systemd}/bin/systemctl => ${lib.getExe' pkgs.systemd "systemctl"}
```

Зависимости: python3

Использование:

```sh
python3 main.py <path>
```

Path — путь до файла или директории. В случае директории заменяет файлы внутри рекурсивно
